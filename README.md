### version 2.1
- 랜덤 프로필 동물 아이콘 변경
- 닉네임 추가 기능
- 플레이어 & 케이스 셔플 기능

- 웹 접근성 지침에 부합하도록 마크업 수정

- 플레이어 수에 따른 랜덤 프로필과 랜덤 사다리 설정
- 사다리 게임 로직을 리액트의 상태 관리와 자바스크립트로 구현
- 캔버스를 활용하여 사다리 경로를 그리기 기능 구현
